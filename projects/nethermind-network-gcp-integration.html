Project Overview

This project sets up an Ethereum testnet using Nethermind nodes orchestrated with Helm charts on Google Cloud Platform. This setup includes a bootnode and multiple mining nodes, providing a robust environment for smart contract development and testing.

It can be replicated exactly as it is - all the way from setting up infra using terraform, to obtaining Eth gas from the Sepolia faucet and testing a sample contract deployment.

Technical Implementation

1. Infrastructure Setup

Deploy the GCP infrastructure using Terraform:

# Set up authentication to GCP
gcloud auth application-default login

# Navigate to the terraform directory
cd terraform

# Initialize Terraform
terraform init

# Review the infrastructure plan
terraform plan

# Deploy the infrastructure
terraform apply

2. Node Configuration

The setup includes two types of Ethereum nodes:

  • Bootnode: Entry point for peer discovery in the Ethereum network
  • Miner Nodes: Responsible for creating new blocks and processing transactions

Network Details:

  • Network ID: 1337
  • Chain ID: 11155111 (Sepolia testnet)
  • Mining Nodes: 2 (configurable)
  • Bootnode: 1

3. Smart Contract Testing

Testing workflow from local development to production:

# Navigate to the nethermind directory
cd nethermind

# 1. Install dependencies
npm install

# 2. Local testing
npx hardhat test

# 3. Testing on GCP cluster (using port forwarding)
./scripts/port-forward.sh  # In one terminal
npx hardhat run scripts/deploy.js --network gcp  # In another terminal

4. Wallet Setup

Setting up MetaMask for Sepolia testnet:

  1. Install MetaMask
  2. Create a new wallet
  3. Connect to Sepolia:
    • Click network dropdown
    • Select "Sepolia Test Network"
    • If not visible: Settings > Advanced > Show test networks

Getting test ETH:

  1. Visit Google Cloud Web3 Faucet
  2. Sign in with your Google Cloud account
  3. Enter your MetaMask address
  4. Request test ETH
